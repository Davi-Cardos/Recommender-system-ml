# Sistema de Recomendação

## 📌 Sobre o projeto
Este projeto implementa um sistema de recomendação utilizando técnicas de filtragem colaborativa com base no dataset MovieLens.

## 🧠 Como funciona
- Cálculo de similaridade entre usuários/itens
- Geração de recomendações personalizadas
- Avaliação com métricas como RMSE

## 🛠️ Tecnologias
- Python
- Pandas
- NumPy

## 📊 Resultados
O sistema foi capaz de gerar recomendações coerentes com base no histórico dos usuários.

## 🎯 Objetivo
Explorar técnicas de sistemas de recomendação e avaliar diferentes abordagens.
