# Sistema de Recomendação (MovieLens)

## 📌 Sobre o projeto
Este projeto implementa um sistema de recomendação utilizando o dataset MovieLens, com o objetivo de sugerir itens com base no comportamento de usuários.

## 🧠 Abordagem
Foram exploradas técnicas de filtragem colaborativa, incluindo:

- User-based KNN
- Model-based (SVD)
- Cálculo de similaridade entre usuários

- Python
- Pandas
- NumPy
